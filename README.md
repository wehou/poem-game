# 诗词背诵小游戏 / Poem Recitation Game

一个帮助三年级学生背诵必背诗词的互动小游戏。

An interactive game to help third-grade students memorize required poems.

## 功能特点 / Features

- **诗词列表模式 / Poem List Mode**: 浏览所有诗词，查看学习进度
- **随机练习模式 / Random Practice Mode**: 根据上下句互问进行背诵练习
- **背诵模式 / Recitation Mode**: 点击显示诗句，逐步背诵
- **熟练度追踪 / Mastery Tracking**: 记录每首诗的熟练程度
- **智能复习 / Smart Review**: 根据错误次数智能推荐需要复习的诗句

## 使用方法 / How to Use

1. 打开 `poem-game.html` 文件
2. 输入你的名字开始学习
3. 选择诗词列表或随机练习模式
4. 点击诗词卡片进入背诵模式
